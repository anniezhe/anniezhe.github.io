# Annie He's Portfolio Website
Welcome to the GitHub repository for my website! This repository contains my website's codebase. The following are frameworks and tools that I've used to build my portfolio website: React.js, MUI, React Native, React Router Dom, and MDBootstrap. To get started, you'll need to perform `git clone` to clone my project in your own computer. Once you have a copy of the codebase, please ensure that you have npm and node installed in your computer. See [here](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) for the instructions on how to check them. From there, see this website [here](https://plainenglish.io/blog/how-to-clone-an-app-from-github-446541a0302d) on how to run my React based website. **Note**: If you would like to share ideas or suggestions on improving my website, please email me [anniezhe0@gmail.com](mailto:anniezhe0@gmail.com).
